User Signup Test

Note: This is a test page for the signup functionality. If you encounter "Server configuration error", it means the Supabase environment variables need to be configured in your deployment platform (Vercel, etc.).

Debug Steps:
  1. Click "Test Database Connection" to check if your environment variables are set correctly
  2. If the test fails, set SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY in your Vercel environment variables
  3. Try signing up with a new email address
